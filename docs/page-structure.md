# 求职助手 — 页面结构设计

> 版本：v1.0  
> 更新日期：2026-06-25  
> 框架：Vue 3 + uni-app + Vue Router + Pinia

---

## 1. 路由结构（Web 端）

### 1.1 路由表

```
/                          →  首页 / Dashboard
/login                     →  登录页
/register                  →  注册页

/resume                    →  简历中心（重定向到 /resume/list）
  /resume/list             →  简历列表
  /resume/create           →  创建简历
  /resume/:id              →  简历详情 / 编辑
  /resume/:id/optimize     →  简历优化（AI 润色）
  /resume/:id/score        →  简历评分
  /resume/:id/versions     →  版本管理

/interview                 →  模拟面试（重定向到 /interview/setup）
  /interview/setup         →  面试设置
  /interview/session/:id   →  面试对话
  /interview/report/:id    →  面试报告
  /interview/history       →  历史记录

/jobs                      →  职位匹配（重定向到 /jobs/list）
  /jobs/list               →  职位列表
  /jobs/:id                →  职位详情
  /jobs/applications       →  投递追踪看板

/plan                      →  求职规划（重定向到 /plan/assessment）
  /plan/assessment         →  能力评估
  /plan/timeline           →  求职时间线
  /plan/tasks              →  每日任务
  /plan/learning           →  学习路径

/company                   →  公司画像（重定向到 /company/list）
  /company/list            →  公司列表
  /company/:id             →  公司详情
  /company/compare         →  公司对比

/prepare                   →  求职准备
  /prepare/questions       →  常见面试题
  /prepare/tech            →  技术题库
  /prepare/exam            →  笔试练习
  /prepare/salary          →  薪资谈判
  /prepare/offer           →  Offer 评估

/rental                    →  租房选址
  /rental/map              →  地图找房
  /rental/compare          →  多公司对比选址
  /rental/city             →  城市对比

/profile                   →  个人中心
  /profile/settings        →  账号设置
  /profile/quota           →  配额管理
```

### 1.2 导航结构

```
┌──────────────────────────────────────────────────┐
│  Logo   简历  面试  职位  规划  公司  准备  租房    │  ← 顶部导航
├──────────────────────────────────────────────────┤
│                                                  │

│  © 2026 求职助手  |  关于  |  帮助  |  反馈        │  ← 底部（Web）
└──────────────────────────────────────────────────┘
```

## 6. 响应式适配策略

| 断点 | 布局 | 说明 |
|------|------|------|
| ≥ 1200px | 双栏 / 三栏 | 桌面端完整布局 |
| 768px - 1199px | 单栏 / 双栏 | 平板端 |
| < 768px | 单栏 + 底部 Tab | 移动端 / 小程序 |

- 地图找房：桌面端使用左右分栏（地图 + 列表），移动端使用上下切换（地图 / 列表 Tab）
- 简历编辑：桌面端分栏（表单 + 预览），移动端全屏表单
- 投递看板：桌面端横向滚动，移动端纵向堆叠
