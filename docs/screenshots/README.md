# 功能截图 Screenshots

本目录存放跃途 LeapPath 的功能截图。

## 如何生成截图

1. 启动项目（参见 [快速开始](../../README.md#-快速开始)）
2. 使用浏览器访问 http://localhost:5173
3. 使用默认账号登录：`demo@leappath.app` / `leappath`
4. 按以下列表截取各模块界面

## 截图清单

| 文件名 | 页面 | 说明 |
|--------|------|------|
| `dashboard.png` | 工作台 | 登录后的首页仪表盘 |
| `resume.png` | 简历中心 | 简历列表页面 |
| `interview.png` | 模拟面试 | 面试会话页面 |
| `jobs.png` | 职位匹配 | 职位列表页面 |
| `company.png` | 公司画像 | 公司列表页面 |
| `plan.png` | 求职规划 | 规划时间线页面 |
| `prepare.png` | 求职准备 | 题库页面 |
| `rental.png` | 租房选址 | 房源地图页面 |

## 截图规范

- 分辨率：1920×1080 或以上
- 格式：PNG
- 主题：使用默认 `leap` 主题截取
- 内容：确保演示数据完整展示
